# 🌾 開心農場

純前端農場模擬遊戲，無任何外部依賴、無需安裝，開啟網頁即可遊玩。

🔗 **線上遊玩：** https://prayer168.github.io/happy_farm/

---

## 遊戲玩法

1. **輸入名字** 開始，進度自動儲存於瀏覽器，下次開啟自動繼續
2. **選種子** → **點農地種植**
3. 定時用 💧 **澆水**（超過澆水間隔會枯萎）
4. 成熟後用 🌾 **收穫**，放入倉庫
5. 點 **賣出全部** 換取金幣、累積經驗值升等
6. 升等解鎖更多農地與高價作物

## 注意事項

| 危險行為 | 結果 |
|----------|------|
| 太久沒澆水 → 枯萎 → 再不澆 | 💀 缺水枯死 |
| 短時間內澆兩次水 | 🫧 溺斃（根部腐爛） |
| 施肥超過 2 次 | ☠️ 肥料中毒死亡 |
| 暴風雨天氣 | ⛈️ 每秒有機率損傷作物 |

## 天氣系統

| 天氣 | 效果 |
|------|------|
| 🌞 晴天 | 正常 |
| ☁️ 多雲 | 澆水間隔延長 15% |
| 🌧️ 下雨 | 每 25 秒自動澆水全部農地 |
| 🌵 乾旱 | 澆水間隔縮短 45%，更快枯萎 |
| ⛈️ 暴風雨 | 每秒 2% 機率損傷作物 |

## 多人遊玩

- 每位玩家輸入不同名字，存檔完全獨立互不干擾
- 同一台電腦可切換玩家（Header 點 👤 按鈕）
- 不同設備各自儲存，天然隔離

## 技術

- 純 HTML / CSS / JavaScript（無框架、無依賴）
- Web Audio API 合成音效（無外部音訊檔）
- localStorage 持久儲存（每位玩家獨立 key）
- CSS Container Queries 實現農場格子完美正方形
- 全 RWD：桌機三欄 / 平板縮放 / 手機垂直堆疊
- GitHub Actions 自動部署 GitHub Pages
